

Provides views filter criteria and contextual filter 
"File Usage: Show files of child tasks".
Filter gets files of project's tasks or files of task's subtasks.

Usage example:
1) add contextual filter "File Usage: Show files of child tasks"
2) Provide default value -> Type -> Content Id from URL
3)  Specify validation criteria -> Validator -> Project, Task
4) Filter value format -> Node ID
5) Save contextual filter
6) add contextual filter criteria "File Usage: Show files of child tasks" 
7) choose filter parameters e.g. expose it
8) have fun
